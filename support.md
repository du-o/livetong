# Live通 技术支持

感谢使用 Live通。

Live通是一款用于查看日本演唱会信息的应用，提供近期演出、全部演出、票务阶段、场馆、城市筛选、心愿单和计划通等功能。

## 常见问题

### 1. 演出信息从哪里来？

Live通会整理公开可访问的演唱会信息来源，包括票务平台、场馆官网等。演出时间、票务状态、购票链接等信息可能随官方页面更新而变化，请以最终购票页面和官方公告为准。

### 2. 为什么有些演出信息不完整？

不同信息源公开的数据字段不同，部分演出可能暂时缺少图片、票务阶段、开场时间或购票平台信息。我们会持续优化数据补全能力。

### 3. 为什么有时无法刷新数据？

请确认手机网络连接正常，并允许 App 使用网络。如果刚授权网络后仍无法加载，可以尝试下拉刷新或重新打开 App。

### 4. 心愿单数据保存在哪里？

心愿单数据保存在你的手机本地。卸载 App 后，本地心愿单数据可能会被删除。

### 5. 如何反馈问题？

如果你在使用过程中遇到问题，或发现演出信息有误，可以通过以下方式联系：

- 邮箱：请填写你的联系邮箱
- GitHub Issues：请填写你的 GitHub Issues 链接

反馈时建议提供：

- 手机型号
- iOS 版本
- App 版本号
- 问题截图
- 相关演出名称或页面
